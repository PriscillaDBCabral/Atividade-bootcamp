# 🚀 Mulheres na Tecnologia: Dorothy Vaughan

Este projeto foi desenvolvido para resgatar e destacar a trajetória de Dorothy Vaughan, uma das principais mentes por trás dos cálculos de engenharia e da introdução da computação eletrônica na exploração espacial.

---

## 👩‍🔬 Quem foi Dorothy Vaughan

Dorothy Jean Johnson Vaughan nasceu em 1910, em Kansas City, e formou-se em matemática com apenas 19 anos. Antes de fazer história na ciência espacial, trabalhou como professora durante os anos difíceis da Grande Depressão. 

Em 1943, durante a Segunda Guerra Mundial, ingressou na antiga NACA (órgão que mais tarde se tornaria a NASA) para trabalhar no Laboratório Aeronáutico de Langley. Por conta das leis de segregação racial vigentes na época, ela foi enviada para a unidade *West Area Computers*, uma ala separada voltada apenas para matemáticas afro-americanas. Em 1949, Dorothy rompeu uma barreira histórica ao se tornar a primeira supervisora negra da agência, onde liderou sua equipe e lutou ativamente por salários mais justos e pelo reconhecimento de suas colegas.

## 🏆 Principais Conquistas e Visão Estratégica

O grande diferencial de Dorothy Vaughan foi sua capacidade de antecipar o futuro da tecnologia. Quando a NASA comprou seus primeiros computadores físicos (as máquinas eletrônicas da IBM), ela percebeu imediatamente que o trabalho das "computadoras humanas" seria extinto. 

Em vez de recuar, ela aprendeu sozinha a linguagem de programação FORTRAN e treinou toda a sua equipe de mulheres negras antes mesmo que os engenheiros homens dominassem as máquinas. Com isso, garantiu que seu grupo liderasse a transição para a computação moderna e tornou-se especialista em programação, contribuindo diretamente para o programa de foguetes de lançamento *Scout*.

---

## 💡 Impacto na Sociedade e Legado

Por décadas, o trabalho de Dorothy permaneceu invisível para o grande público, ganhando o devido reconhecimento mundial apenas com o livro e o filme *Estrelas Além do Tempo*. 

Seu impacto vai muito além das missões espaciais bem-sucedidas. Como mentora e líder, ela abriu caminho para que gerações de minorias e mulheres ocupassem cargos de engenharia e alta tecnologia. A transição que ela comandou — da matemática no papel para o código de computador — ajudou a moldar as bases da programação que usamos até hoje.

# Aprendizados sobre Git e GitHub

Este espaço foi reservado para documentar os conceitos fundamentais e comandos práticos absorvidos durante o gerenciamento deste projeto.

---

## Entendendo as Ferramentas

Ficou clara a diferença entre o papel de cada uma dessas tecnologias na rotina de desenvolvimento:

* **Git:** É o sistema de controle que roda diretamente no computador. Ele funciona como uma máquina do tempo, rastreando as alterações nos arquivos e permitindo criar históricos de salvamento do projeto.
* **GitHub:** É a plataforma em nuvem onde hospedamos esses repositórios. Ele serve para guardar os projetos online, compartilhar o código e facilitar o trabalho em equipe.

---

## Comandos Essenciais

Abaixo estão os principais comandos utilizados para organizar e salvar o progresso do trabalho de forma segura:

* **git init:** Cria um repositório Git local dentro da pasta do projeto, iniciando o rastreamento dos arquivos.
* **git clone:** Faz o download e copia um repositório que já existe no GitHub para o computador.
* **git add:** Seleciona e prepara os arquivos modificados para o próximo salvamento (coloca as alterações na área de *staging*).
* **git commit -m "mensagem":** Salva permanentemente as alterações preparadas no histórico do projeto, acompanhado de uma mensagem breve sobre o que foi feito.
* **git push:** Envia todos os salvamentos (commits) feitos no computador para o repositório online no GitHub.
